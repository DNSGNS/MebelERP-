## О проекте

**FurniFlow** — кроссплатформенное мобильное приложение класса ERP для малого и среднего мебельного производства. Система охватывает полный жизненный цикл заказа: от первичного замера на объекте клиента до финансового отчёта после установки.

Каждый сотрудник работает в своём ролевом интерфейсе — замерщик, кладовщик, цеховик, менеджер — и видит только то, что нужно именно ему.

## Роли и интерфейсы

### 📐 Замерщик
Основной инструмент для сбора данных непосредственно на объекте клиента.

- **Список проектов** — все активные замеры пользователя, создание нового проекта
- **Детализация проекта** — многовкладочный интерфейс:

| Вкладка | Содержимое |
|---|---|
| ЛДСП | Детали: длина, ширина, количество |
| Фасады | Тип, размеры, фрезеровка, цвет |
| Двери / Профили | Количество дверей, параметры вставок |
| Мягкая часть | Тип мягкого места, щит, прочие данные |
| Фурнитура | Петли, направляющие, подъёмные механизмы |
| Отчёт | Итоговая сводка, предварительная стоимость |

Система автоматически генерирует **карты раскроя** и рассчитывает предварительную стоимость с учётом наценок.

---

### 🏭 Кладовщик
Контроль комплектации перед запуском в производство.

- Отображаются только проекты со статусом **«Новый»**
- Просмотр полного списка материалов и фурнитуры
- Чекбоксы готовности — при отметке обоих проект уходит в **«В работе»** и задания появляются у цеховиков

---

### ⚙️ Цеховик
Панель производственных задач по технологическим этапам.

| Вкладка | Описание |
|---|---|
| **Раскрой** | Задачи по распилу ЛДСП, статусы «взял» / «готов» |
| **Раскладка (МДФ)** | Задачи МДФ, бронирование и отмена |
| **Двери / Профили** | Задачи по дверям и профилям |
| **Кромка** | Список деталей для кромления |

---

### 👔 Менеджер
Управление процессами, логистикой и финансами.

- **Текущие проекты** — активные заказы с датами, назначением установщиков
- **Завершённые проекты** — архив с деталями фурнитуры и фотоотчётами

```
[Замер] → [Новый] → [Комплектация] → [В работе] → [Производство] → [Готов к установке] → [Выполнен]
```

## Технологии

| Компонент | Технология |
|---|---|
| Мобильный клиент | .NET MAUI (Android) |
| Серверная часть | ASP.NET Core Web API |
| База данных | PostgreSQL + Entity Framework Core |
| Графика раскроя | SkiaSharp |
| Экспорт отчётов | QuestPDF / ClosedXML |
| Обмен данными | REST API + JSON + HTTPS |

---

## Ключевые возможности

- Автоматическая генерация карт раскроя (до 5 сек. на изделие)
- Сквозное отслеживание статусов без дублирования данных
- Ролевое разграничение доступа — финансы видит только менеджмент
- Автоматический расчёт сдельной зарплаты на основе выполненных заданий
- Офлайн-режим для замерщика при отсутствии интернета
- Экспорт финансовой аналитики в PDF и Excel
